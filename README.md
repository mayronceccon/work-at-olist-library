# Installation
```bash
git clone git@github.com:mayronceccon/work-at-olist-library.git

cd work-at-olist-library

python -m venv .venv

source .venv/bin/activate

pip install -r requirements.txt

cp .env.example .env

python manage.py migrate
```

# Test
```
python manage.py test
```
